# Dice Game

A simple interactive dice game built with HTML, CSS, and JavaScript.

## Features
- Roll two dice simultaneously
- Compare results to determine the winner
- Fresh, modern UI design
- Responsive layout

## Files
- `dicee.html` - Main HTML structure
- `index.js` - Game logic and interactivity
- `styles.css` - Styling
- `images/` - Dice images (dice1.png through dice6.png)

## How to Play
1. Open `dicee.html` in your web browser
2. Click the "Refresh Me" button to roll the dice
3. The game will display which player won or if it's a draw
